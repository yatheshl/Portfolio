Q-Learning Taxi-V3
Github
In this project, I created an AI agent to solve the Taxi-v3 environment from OpenAI Gym. The agent's goal is to pick up a passenger at a certain location and drop them off at another location in the shortest amount of time. The agent uses Q-learning, a model-free reinforcement learning algorithm, to learn the optimal actions to take in each state and reduce the number of steps taken to complete the task.

Project Outcomes:
The AI agent was trained over 50,000 episodes and tested on 100 episodes. The average score over the test episodes was calculated to measure the agent's performance. The agent successfully learned to complete the task efficiently, showcasing the effectiveness of Q-learning in solving reinforcement learning problems.
Libraries Used
NumPy: A popular library for numerical computations in Python. Used for creating and manipulating the Q-table.
Gym: OpenAI's library for creating and interacting with reinforcement learning environments.
Random: Python's built-in library for generating random numbers. Used for exploration-exploitation trade-off.
Matplotlib: A plotting library in Python. Used for visualizing the environment.

Skillset Displayed
  • Reinforcement Learning: Implemented the Q-learning algorithm to train an AI agent to solve the Taxi-v3 environment.

  • Exploration-Exploitation Trade-off: Balanced the exploration and exploitation during the learning process by implementing the epsilon-greedy strategy.

  • Hyperparameter Tuning: Tuned learning rate, discount factor, and exploration rate decay to optimize the learning process.

  • Environment Interaction: Utilized the Gym library to interact with the Taxi-v3 environment, executing actions, and receiving rewards and state transitions.

  • Visualization: Visualized the environment during testing using Matplotlib to display the agent's progress in real-time.
