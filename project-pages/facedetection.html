Real-Time Face Recognition
Github
This project presents a real-time face recognition and identification system using OpenCV, face_recognition library, and Support Vector Machine (SVM) classifier. The system is designed to recognize and identify known individuals from a video stream, displaying their names and the confidence level of the recognition.

Key Features:
  • Preprocessing and Encoding of Known Faces: The system preprocesses a set of known face images stored in the known_faces directory, generating face encodings for each person. The face encodings are then averaged to create a single representation for each individual.

  • Training an SVM Classifier: A linear SVM classifier is trained using the face encodings as input features and the names of the individuals as target labels. This classifier is responsible for predicting the identity of a person from their face encoding.

  • Real-Time Face Recognition: The system captures video frames using OpenCV's VideoCapture object, resizing and converting them to the appropriate color space. Face locations are detected using the face_recognition library, and encodings are generated for each detected face.

  • Identification and Confidence Level Estimation: The SVM classifier predicts the identity of each detected face, along with the corresponding confidence level (probability). The system displays the recognized individual's name and confidence level as an overlay on the video frame.

  • Display and User Interaction: The video stream, along with the recognition and identification results, is displayed in real-time using OpenCV's imshow function. The system allows the user to stop the video stream by pressing the 'q' key.
